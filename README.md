# Marina Towers Rentals

A luxury rental marketplace built with Next.js, Tailwind CSS, and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

Push to GitHub and import at [vercel.com/new](https://vercel.com/new).

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **TypeScript**
